Discord Setup Guide

Everything you need to join the server and get set up fast.

A clear onboarding page for new members: create an account, join the server, verify access, choose the right channels, and fix common setup problems.

Start setup
Step 1

Create an account

If you are new to Discord, click this link, and create an account.

https://discord.com/register

Choose a username

Verify your email before trying to join the server.

And use a strong password.

Step 2

Log in

If you already have an account, sign in through the Discord app or website.

Use the same account you want connected to the server.

Reset your password first if login is failing.

Optional: Turn on two-factor authentication for another layer of saftey.

Step 3

Join the server

Open the invite link, accept the invitation, and make sure you check out the rules channel.

https://discord.gg/dubeR8r3Te

If you need to log in again please do so.

If the invite is invalid for some reason contact emmett.austin@pcc.edu,

Step 4

Find the right channels

Share some pictures of your pets

Online Labs Folder

Support

Troubleshooting

Can’t create an account: check for email verification messages or try another browser.

Can’t log in: use password reset and confirm you are on the correct account.

Can’t see channels: complete verification or contact a moderator.

Notifications are overwhelming: update server notification settings after joining.

Need help?

Add your actual support contact here, such as a moderator name, help email, or a link to the server’s support channel so people know where to go if setup fails.
